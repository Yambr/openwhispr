## Caveat / build-env note

The first run failed the value assertion (`Expected https://auth.openwhispr.com,
got http://localhost:4000`) because the local root `.env` sets
`OPENWHISPR_AUTH_URL=http://localhost:4000`, which Vite bakes into the dist. The
test hardcodes the upstream default. Rebuilding the renderer with the AUTH_URL
override cleared (`env OPENWHISPR_AUTH_URL= npm run build:renderer`) made the
default-host assertion pass. CI builds without that local `.env`, so this is a
local-dev artifact, not a code issue.
