<interfaces>
The 3 pre-existing tests use Node's built-in test runner:
```js
const test = require("node:test");
const assert = require("node:assert/strict");
test("name", () => { assert.equal(a, b); });
```
Vitest equivalent (with `globals: true` in vitest.config.ts):
```js
const { test, expect } = require("vitest");
test("name", () => { expect(a).toBe(b); });
```
OR with globals enabled, drop the require entirely.

Vitest is fully compatible with `require()` for CommonJS sources (the helpers under test are CJS). No ESM conversion needed for the test files themselves; vitest auto-handles both.
</interfaces>
</context>
