Adopt upstream OpenWhispr PR #835 (commit `69cb74be`, "fix: skip API key check for self-hosted transcription servers").

Purpose: When a user runs a self-hosted transcription server (transcriptionMode === "self-hosted" with a configured remoteTranscriptionUrl), no cloud API key is required. Without this guard, getAPIKey() forces a key lookup that self-hosted users do not have, blocking transcription.

This is straight upstream adoption — the helper and guard match upstream verbatim (upstream-parity, minimize merge cost); only the test harness is adapted (node:test → vitest).
